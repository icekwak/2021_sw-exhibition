## 주제
2021 씨애랑 온라인 소프트웨어 전시회 홈페이지

## 팀원
곽영주 : 백엔드<br>
함승완 : 프론트엔드

## 개요
코로나로 인해 작년 SW전시회를 제대로 못하고 깃허브 주소로만 작품을 전시했던 것을 이번에는 작년 처럼이 아닌 SW전시회 웹사이트를 제작하여 좀 더 좋은 환경에서 사람들이 작품을 보면 좋겠다 싶어서 제작하였습니다.
